Guía paso a paso para implementar el sistema de traducción en un nuevo proyecto:

Paso 1: Copia el archivo `lang.js`
Copia el archivo `js/lang.js` de tu proyecto actual y pégalo en la carpeta de JavaScript (por ejemplo, `js/`) de tu nuevo proyecto.

Paso 2: Adapta el archivo `lang.js`
Abre el archivo `lang.js` que acabas de copiar y modifica el objeto `translations` para que contenga los textos de tu nuevo sitio web. Deberás definir tus propias claves y los textos correspondientes en español e inglés.

Ejemplo de cómo debería verse el objeto `translations`:
```javascript
const translations = {
  es: {
    // Nuevas traducciones en español
    titulo_principal: "Bienvenido a mi Nuevo Sitio",
    parrafo_intro: "Este es un párrafo de introducción."
  },
  en: {
    // Nuevas traducciones en inglés
    titulo_principal: "Welcome to my New Site",
    parrafo_intro: "This is an introductory paragraph."
  }
};
// El resto del código en lang.js no necesita cambios.
```

Paso 3: Modifica tu HTML para la traducción
En tu archivo HTML, añade el atributo `data-i18n` a cada elemento que quieras que se traduzca. El valor de este atributo debe ser una de las claves que definiste en el paso anterior.

Ejemplo en tu HTML:
```html
<h1 data-i18n="titulo_principal">Bienvenido a mi Nuevo Sitio</h1>
<p data-i18n="parrafo_intro">Este es un párrafo de introducción.</p>
```

Paso 4: Agrega el botón para cambiar de idioma
Inserta un botón en tu archivo HTML. Es muy importante que este botón tenga exactamente el `id="langSwitch"`, ya que el script lo usa para funcionar.

Ejemplo del botón en tu HTML:
```html
<button id="langSwitch" aria-label="Change language">
  EN
</button>
```

Paso 5: Incluye el script en tu HTML
Finalmente, para que todo funcione, asegúrate de que tu archivo HTML esté cargando el script `lang.js`. La mejor práctica es añadir la etiqueta `<script>` justo antes del cierre de la etiqueta `</body>`.

Ejemplo de cómo incluir el script en tu HTML:
```html
<!DOCTYPE html>
<html lang="es">
<head>
    <!-- ... -->
</head>
<body>
    <!-- Todo el contenido de tu página -->

    <script src="js/lang.js"></script>
</body>
</html>
```
