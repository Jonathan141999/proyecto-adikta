# Adikta — Proyecto web

Somos una empresa nacional que se lanza al mercado internacional bajo la marca **Adikta**.

Resumen del proyecto:
- Sitio web estático con páginas principales: `index.html`, `productos.html`, `servicios.html`, `nosotros.html`, `contacto.html`.
- Componentes reutilizables en `components/` (barra de navegación y pie de página).
- Estilos en `assets/css/` y scripts en `assets/js/`.
